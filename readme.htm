zpBootstrap – a ZenPhoto/ZenPage theme

zpBootstrap is a « Responsive » theme for ZenPhoto.

Description

Important: To use the release 1.4.2 of the theme, you must have ZenPhoto 1.4.2 (or more).

Scripts used:

  • Bootstrap (HTML, CSS, and JS toolkit for Responsive WebSite)
  • Flexslider (a fully responsive jQuery slider plugin)
  • AddThis (snippet to add sharing tools to your site)
  • colorbox (lightweight customizable lightbox plugin for jQuery)
  • script for navigation with the arrow keys.

The theme supports the following ZenPhoto plugins:

  • comment_form, contact_form, dynamic-locale, flag_thumbnail, rating, register_user, user_login-out, zenpage

The theme does not support the following ZenPhoto plugins :

  • colorbox and slideshow (zpBoostrap uses and configures it's own colorbox and modal plugins)
  • GoogleMap and EXIF (not enough time to do that in this first release)

Installation

  1. Upload the zip file to your computer,
  2. Unzip the downloaded zip file locally, and upload the zpBoostrap folder to the directory /themes/ of your ZenPhoto site,
  3. In ZenPhoto administration, go to the Themes tab and activate the zpBootstrap theme,
  4. Navigate to Options>Theme to view and configure the available options for zpBootstrap.

Options

  1. You can display a home page, with a slider of 5 random picts, the gallery description and the latest news (if zenpage is used),
  2. Only one RSS Feed is displayed: go to options>RSS and select the RSS feed to use (RSS Feed "All News" has priority over RSS Feed "Gallery").

For the theme works well, I advise you

  • To enter the title and url of your website, the title and description of your gallery in admin>options>gallery.
  • To make responsive images in news and pages.
    To do that, edit the html source of news and pages and add class="remove-attributes" on each image (the result should be < img class="remove-attributes" src="the_path_to_your_image" />)

 

English version - Version Française

zpBootstrap – un thème ZenPhoto/ZenPage

zpBootstrap est un thème « Responsive » pour ZenPhoto.

Description

Important : Pour utiliser la version 1.4.2 du theme, vous devez utiliser ZenPhoto 1.4.2 (ou plus).

Scripts utilisés :

  • Bootstrap (Framework HTML, CSS, et JS pour construire des sites « Responsives »)
  • Flexslider (plugin jQuery pour faire des diaporamas « Responsives »)
  • AddThis (code pour ajouter des outils de partage à votre site)
  • colorbox (plugin jQuery pour faire des lightbox)
  • navigation avec les flèches du clavier sur les actualités et les images.

Le thème supporte les plugins ZenPhoto suivants :

  • comment_form, contact_form, dynamic-locale, flag_thumbnail, rating, register_user, user_login-out, zenpage

Le thème ne supporte pas les plugins ZenPhoto suivants :

  • colorbox et slideshow (zpBoostrap utilise et configure ces propres plugins colorbox et modal)
  • GoogleMap et EXIF (pas assez de temps pour les embarquer dans cette première version)

Installation

  1. Téléchargez le fichier zip sur votre ordinateur,
  2. Décompressez le fichier zip localement, et télécharger le dossier zpBoostrap dans le répertoire /themes/ de votre site ZenPhoto,
  3. Dans l'administration ZenPhoto, allez sur l'onglet Thèmes et activer le thème zpBoostrap,
  4. Allez sur Options>Thème afin d'afficher et configurer les options disponibles pour zpBoostrap.

Options

  1. Vous pouvez afficher une page d'accueil, avec un diaporama de 5 images aléatoires issues de la galerie, la description de la galerie et la dernière actualité,
  2. Un seul flux RSS est affiché : allez sur Options>Flux RSS et choisissez le flux à utiliser (le flux "Toutes les actualités" est prioritaire sur le flux "Galerie").

Pour que le thème fonctionne bien, je vous conseille

  • d'entrer le titre et l'url de votre site, le titre et la description de la galerie dans admin>options>galerie,
  • de rendre flexibles les images utilisées dans les actualités et les pages.
    pour ce faire, modifiez le code html de vos actualités et de vos pages et ajoutez class="remove-attributes" sur chaque image (le résultat doit être < img class="remove-attributes" src="the_path_to_your_image" />)
